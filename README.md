
# React Redux Typescript Bitcoin Prices 

A React, Redux and Typescript seed application.

Built by rjmacarthy: https://www.cryptogrind.com

## Installation

```
  npm install 
  typings install
```

## Usage


Build application

```
npm run build
```

Run development server with webpack

```
npm run dev
```
